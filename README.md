#### 🚄 [FastRail — Railway Reservation System](https://github.com/Rishi006knight/FastRailapp)

A full-stack railway booking platform with real-time seat search, ticket booking, booking history, and a full admin dashboard with analytics.

**Highlights**
- 🔍 Search & book trains with live seat availability
- 👤 JWT-secured auth for users and admins
- 📊 Admin dashboard — revenue charts, booking breakdown, train management
- 🗄️ Seeded with real Indian railway routes (Rajdhani, Shatabdi, Vande Bharat, and more)

**Stack:** React 18 · TypeScript · Tailwind CSS · React Router · Spring Boot 3 · MongoDB · JWT

🔗 **Live:** [fast-railapp.vercel.app](https://fast-railapp.vercel.app) &nbsp;|&nbsp; **Code:** [Rishi006knight/FastRailapp](https://github.com/Rishi006knight/FastRailapp)
